# Tello 顔検知ドローン制御システム

## 概要

DJI Tello と OpenCV を使用した顔検知対応ドローン制御プログラムです。

本システムでは、

- 顔検知（正面顔・横顔）
- 肌色・髪色フィルタによる誤検知低減
- 写真撮影
- 動画録画
- キーボードによる飛行制御

を実装しています。

---

## 使用ライブラリ

- Python
- OpenCV
- djitellopy
- pygame

---

## インストール

```bash
pip install opencv-python
pip install pygame
pip install djitellopy
```

---

## 起動方法

```bash
python tello_face_detection.py
```

---

## 操作方法

| キー | 動作 |
|--------|--------|
| Space | 離陸 / 着陸 |
| ↑ ↓ | 上昇 / 下降 |
| ← → | 左右回転 |
| E / C | 前進 / 後退 |
| S / F | 左移動 / 右移動 |
| J長押し(3秒) | 写真撮影 |
| U長押し(3秒) | 録画開始 / 停止 |
| ESC | 終了 |

---

## 主な機能

### 顔検知

Haar Cascade を利用して

- 正面顔
- 左向き横顔
- 右向き横顔

を検出します。

### 誤検知対策

以下の手法を実装しています。

- 肌色判定
- 髪・眉・目の暗部判定
- 顔追跡による連続検出確認
- 重複顔枠統合

### 撮影機能

- 写真保存
- 動画録画
- 自動ファイル名生成
